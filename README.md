# haicode

Root language system for Haicode and the hai* programming family.

## Scope

Doktrin Haicode, keluarga hai*, Converhai, dan arah bahasa pemrograman JGS.

## Relationship To JGS

This repository is part of the JuGloW Global System. JGS is the central documentation hub that links to this repository and other separate ecosystem packages.

- JGS hub: https://github.com/JuGloW/JGS
- Planned/active repo: https://github.com/JuGloW/haicode

## Repository Rule

This repository should contain only its own focused source, docs, tests, and package data. It should not become a bundle for unrelated JGS packages.
